Stuff I use for college as a Freshman CS Major

Cloud Storage, Office Suite, Notes, Reminders, Calendar

Default Google suite tools

Drive, Docs, Keep Notes, and GCal all work perfectly fine. Nothing fancy needed.

Text Editor

VSCodium/vim

VSCodium is like vscode but without all the MS telemetry. Vim is my go to for anytime I need to ssh into the school server for class assignments.

Laptop/Computer(s)

Thinkpad T430, MBP 13 2017

98% of the time I use my Thinkpad but there are times when I don't want to unplug it from my monitor and keyboard to bring into bed. It's like almost a decade old, but like we won't need to do any heavy compilation or anything for my classes. I'm doing python now and probably C, Java, and bash for next year. I do recommend using a Linux Distro though, my Thinkpad runs Arch but you could use literally any other distro.

SSH

I just use the ssh command

Just about every OS/Distro should have ssh built in. MacOS most likely has it and windows ships with Openssh now or something. Linux is a no brainer so yeah just know how to use ssh and scp and you're set.

P.S. Go look at the github student pack, they got some goodies if you can provide them with a school email. :D

That's kinda it. You don't really need anything too fancy. Just free some cloud tools and a laptop is all I've needed so far.
